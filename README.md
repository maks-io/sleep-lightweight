Lightweight helper function to wait x milliseconds asynchronously.

## Install

```
npm install --save sleep-lightweight
```

Or via Yarn:

```
yarn add sleep-lightweight
```

## Usage

```typescript
import { sleep } from "sleep-lightweight";

await sleep(5 * 1000) // wait for 5 seconds
```
